# IMPORTANTE

## Esse projeto utiliza json-server, sendo necessário baixar as dependências dele

1- Primeiro baixe o json server por meio do comando *'npm install json-server'*
2- Logo após, inicie o projeto com o comando *'npx json-server db.json'*
3- Esse comando irá expor os endpoints da aplicação, podendo ser utilizados por meio do *Insomnia* ou *URL*
